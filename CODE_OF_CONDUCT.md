# Code of Conduct

This project expects professional, good-faith collaboration.

## Expected Behavior

- Be respectful and direct.
- Keep technical criticism focused on the work.
- Assume mistakes can be corrected through clear evidence and review.
- Respect maintainers' time by keeping issues and pull requests actionable.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Dismissive or hostile comments based on identity or background.
- Repeated off-topic demands after maintainers have given a decision.
- Publishing private security details before the security process has had time
  to respond.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests;
block users; or report severe conduct issues to the hosting platform.

Report security issues through `SECURITY.md`. Report conduct issues privately
to the repository owner.
